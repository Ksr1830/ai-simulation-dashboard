Welcome to the AI Simulation Toolkit!

How to Run:
1. Open Command Prompt.
2. Navigate to the folder: `cd C:\AI_Simulation_Package`
3. Run the batch file: `run_simulation.bat`
4. Your dashboard will launch at `http://localhost:8501`

Notes:
- Python 3.9+ required
- Run `pip install -r requirements.txt` if needed
